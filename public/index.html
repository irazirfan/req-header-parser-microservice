Request Header Parser

User Story:

I can get the IP address, language, and software that I'm currently using.

Example Usage:

https://github.com/irazirfan/req-header-parser-microservice/api/whoami

Example Output:


    {
      ipaddress: "127.0.0.1",
      language: "en-US",
      software: "Macintosh; Intel Mac OS X 10_11_6"
    }
